Sales Analytics Dashboard (SQL | Power BI | DAX)

This project delivers an end-to-end Sales Analytics Solution using SQL, Power BI, and DAX. It analyzes global sales data to uncover insights on revenue, profit, sales trends, regions, and product performance.
